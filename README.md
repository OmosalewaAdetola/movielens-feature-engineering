# MovieLens Feature Engineering and Exploratory Data Analysis

## Overview
This project explores the MovieLens dataset through feature engineering and exploratory data analysis (EDA), 
with the goal of building a clean, analysis-ready dataset that can support future recommendation system modeling.

## Objectives
- Merge multiple datasets into a single unified analytical dataset
- Engineer new features to enrich the data
- Conduct exploratory data analysis to uncover patterns and insights

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Dataset sourced from [GroupLens](https://grouplens.org/datasets/movielens/). 
It contains over 100,000 movie ratings across four related files — ratings, metadata, tags, and movies.

## What Was Done
- Cleaned and merged four datasets (100K+ records) into one unified analytical dataset
- Engineered nine new features including release year, movie age, popularity score, genre count, and rating-based metrics
- Conducted EDA with six visualizations covering rating distribution, genre performance, popularity vs. quality, and rating trends over time
- Delivered a final dataset with 15 analytical variables ready for downstream modeling

## Key Insights
- Rating distribution and genre performance vary significantly across decades
- Popularity does not always correlate with quality
- Certain genres consistently outperform others in average ratings
